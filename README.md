# 🚀 Getting Started

## 1️⃣ Create an Account

- Sign Up
- Login to your account

---

## 2️⃣ Configure Your Profiles

Go to

**⚙️ Settings**

Add your

- 🟢 Codeforces Handle
- 🟡 LeetCode Username
- 🔴 AtCoder Username

Click **Save**.

---

## 3️⃣ Install Chrome Extension

Download the official extension

👉 **Automation Extension**

[https://github.com/Varadharaj-S/automation-extension](https://github.com/Varadharaj-S/Automation_Extension)

After installing

- Login to LeetCode
- Open the Extension
- Click **Connect**

Your LeetCode session will be securely connected.

---

## 4️⃣ Import Your LeetCode History

Open

📊 Dashboard

Click

**Import Full LC**

The system will automatically import all your previous LeetCode submissions.

After the import finishes

✅ Import button becomes disabled

Future submissions are synchronized automatically.

---

## 5️⃣ Automatic Synchronization

Every sync automatically updates

- ✅ Codeforces
- ✅ LeetCode
- ✅ AtCoder
- ✅ Google Sheets

No manual work required after the first setup.
